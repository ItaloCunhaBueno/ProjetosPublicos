upload Envio de Arquivos upload
Clique ou arraste arquivos aqui para enviar.
  • Arraste arquivos para a área de transfêrencia ou clique nela para selecionar o arquivo a ser enviado.
  • O host da página receberá os arquivos na pasta Downloads.
  • É possível enviar multiplos arquivos ao mesmo tempo, para isto, basta selecionar mais de um arquivo.
  • O tamanho máximo de arquivo foi configurado para 50 gigabytes.
  • Os arquivos não são guardados em nuvem, são enviados diretamente para o host.
  • O servidor é executado na rede interna, portanto, é preciso estar na mesma rede para acessar esta página e enviar arquivos.
  • ATENÇÃO: devido a um erro em como o componente de upload lida com a barra de progresso, a mesma pode aparecer completa antes do termino do envio do arquivo. Para assegurar que o envio foi completado, espere até que a barra de progresso desapareça.
